Learn basic JavaScript & jQuery terms or concepts!

Examples:

  • JavaScript

    • JavaScript is an interpreted client-side scripting language that allows a web designer the ability to insert code into their web page.

  • Function

    • It's a group of instructions, also known as a named procedure, used by programming languages to return a single result or a set of results.

  • Method

    • It's a function that belongs to a class or object. It is "bound" to that class or object, defined only within that scope of usage.

  • Variable

    • It's a named unit of data that is assigned a value. If the value is modified, the name does not change.

  • jQuery

    • It's a JavaScript script package meant to make common and repetitive tasks in JavaScript with HTML short and understandable code.

The definitions are attributed to Computer hope.
